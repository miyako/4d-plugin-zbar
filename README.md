* Supported barcode types

```
@since 0.11: CODE93, CODABAR, DATABAR
@since 0.10: QRCODE
@since 0.4: ISBN10, ISBN13, I25, CODE39
@since 0.1: EAN2, EAN5, EAN8, UPCE, UPCA, EAN13, COMPOSITE, CODE128
```

**Caution**: The syntax has changed in ``2.0.0``!
